perf-003 CRÍTICO GSAP se carga después de los scripts que lo usan — orden roto ▶
Problema
gsap.min.js está en línea 2447 pero script.js (línea 2443), cert-gallery.js (2445) y photo-vfx.js (2446) se cargan antes. Errores en runtime silenciados por load order.
Fix
Orden correcto: browser-detect → splitting → scroll-out → gsap → ScrambleTextPlugin → script.js → resto. Además agregar defer a todos los scripts.
ux-001 CRÍTICO prefers-reduced-motion ignorado en styles.css — 58 animaciones activas para usuarios con fotosensibilidad ▶
Fix
Agregar al final de styles.css: @media (prefers-reduced-motion: reduce) { .scanlines, .vhs-static, .orb-wrap, #intro::before, #about::before, .glitch-text [data-char]::after { animation: none !important; } }
arch-001 ALTO Patrón static-line canvas copy-paste 4 veces en script.js (~350 líneas duplicadas) ▶
Fix
Extraer function makeStaticLine(canvasEl, opts) que retorne { show, hide, update }. Reduce ~350 líneas a ~80.
