## 🚀 CÓMO USAR RÁPIDAMENTE

### Si quieres Refactorizar tu Portafolio:

```bash
# 1. Lee la guía del skill
cat SKILL_PORTFOLIO_REFACTOR_EXPERT.md

# 2. Cuando tu portafolio esté listo
cd /ruta/a/tu/portafolio
claude-code portfolio-refactor-expert analyze --repo-path . --agentes all

# 3. Revisa presentación
open refactor-analysis/presentation.html

# 4. Aprueba cambios en panel interactivo
# (El skill te guía)

# 5. Ejecuta refactor
claude-code portfolio-refactor-expert execute --changes-file approved-changes.json
```

### Si quieres Auditar tu Proyecto de Maestría:

```bash
# 1. Lee el README
cat README_AUDITORIA_MAESTRIA.md

# 2. Abre la auditoría
open AUDITORIA_MAESTRIA_PROYECTO_WEB.html

# 3. O ejecuta desde terminal
chmod +x run-audit.sh
./run-audit.sh --project-path . --output ./audit-report

# 4. Lee cada sección
# (20-30 minutos total)

# 5. Implementa recomendaciones por semana
# (Siguiendo el plan de acción)
```
