Option B · Headless CMS (Most Scalable)
portfolio/index.html          Contentful / Sanity API
┌─────────────────────┐       ┌──────────────────────────┐
│  Blog section       │◀─────▶│  CMS API (REST/GQL)      │

PROS:  auto-updating, no HTML edits ever
CONS:  API key mgmt, async fetch logic, rate limits
TIME:  1–2 days to integrate + test

VERDICT: ──────────────────────────────────────────
  Start with Option A. If you post weekly for
  a month, migrate to Option B. Don't over-engineer
  a blog you haven't validated yet.
